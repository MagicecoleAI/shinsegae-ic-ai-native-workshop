# Shinsegae I&C AI Native Workshop

서울대학교 EPM 과정에서 진행하는 신세계 I&C SLP 3기 대상 `AI Native EPM (AI4PM)` 실행 과정 GitHub Pages입니다.

과정은 2026년 6월 26일 인사이트 강의와 팀빌딩에서 시작해, 8월 21일 프로젝트 구현 및 멘토링, 8월 28일 프로젝트 발표 및 회고로 이어집니다.

## 구성

- `index.html`: 3일 과정 흐름을 담은 단일 페이지
- `assets/styles.css`: 반응형 페이지 스타일
- `assets/shinsegae-seminar-cover.png`: 6/26 강의자료 표지 이미지
- `outputs/seminar-20260626.pdf`: 강의자료 PDF
- `outputs/shinsegae-ai-economy-2026-briefing.pdf`: AI Economy 2026 시장 브리핑 PDF
- `outputs/ev-state-of-ai-economy-2026.pdf`: Exponential View AI Economy 2026 원본 보고서 PDF
- `outputs/shinsegae-ai-economy-2026-briefing.pptx`: AI Economy 2026 시장 브리핑 PPTX 원본
- `outputs/shinsegae-ai4pm-0626-3h-workshop.pdf`: 6/26 워크샵 진행자료 PDF
- `outputs/shinsegae-ai4pm-workshop-tracker.xlsx`: Google Sheets로 가져와 사용할 워크샵 운영 스프레드시트
- `outputs/team-01-s2d-profit-management-automation.pdf`: 1조 S2D 프로젝트 예상손익 관리 자동화 발표자료
- `outputs/team-02-weekly-report-automation.pdf`: 2조 주간보고서 자동화 발표자료
- `outputs/team-03-infra-request-guide-ai-agent.pdf`: 3조 인프라 업무 요청 가이드 AI Agent 구현 발표자료
- `outputs/team-04-market-news-stock-disclosure-report-agent.pdf`: 4조 MARK 뉴스·주가·공시 리포트 에이전트 발표자료
- `outputs/team-05-ai-agent-project-management.pdf`: 5조 AI Agent 기반 프로젝트 WBS 관리 발표자료

## 핵심 흐름

1. 6월 26일: AX Native 조직 구성 발제, 사전 진단 공유, 팀빌딩, 8월 후속 과정 안내
2. 8월 21일: 스파로스 데브엑스 기반 프로젝트 구현 및 멘토링
3. 8월 28일: 8시간 템플릿 기반 발표 패키징, 팀별 발표, 회고, 현업 적용 인사이트 도출

## 운영 원칙

- 사전 설문과 참가자 명단은 공개 페이지에 원문으로 노출하지 않고 집계 인사이트만 반영합니다.
- 15명은 3인 1팀, 총 5개 프로젝트팀으로 구성하고 각 팀은 업무 오너, 구현 담당, 검증·발표 담당을 둡니다.
- 팀 주제는 서비스 운영, 업무 시스템, 클라우드/인프라, AX/R&D, 품질/준법/프로젝트 관리 축에서 실제 구현 가능한 현업 문제로 좁힙니다.
- 개별 외부 유료 AI 툴 결제를 전제로 하지 않고 사내 AI 플랫폼인 스파로스 데브엑스를 기본 실습 환경으로 둡니다.
- 결과물은 단순 학습 완료가 아니라 현업에서 어떤 업무를 어떻게 처리할지에 대한 실행 인사이트입니다.
