# Shinsegae I&C AI Native Workshop

서울대학교 EPM 과정에서 진행하는 신세계 I&C SLP 3기 대상 `AI Native EPM (AI4PM)` 실행 과정 GitHub Pages입니다.

과정은 2026년 6월 26일 인사이트 강의와 팀빌딩에서 시작해, 8월 21일 프로젝트 구현 및 멘토링, 8월 28일 프로젝트 발표 및 회고로 이어집니다.

## 구성

- `index.html`: 3일 과정 흐름을 담은 단일 페이지
- `assets/styles.css`: 반응형 페이지 스타일
- `assets/shinsegae-seminar-cover.png`: 6/26 강의자료 표지 이미지
- `outputs/seminar-20260626.pdf`: 강의자료 PDF

## 핵심 흐름

1. 6월 26일: AX Native 조직 구성 발제, 사전 진단 공유, 팀빌딩, 8월 후속 과정 안내
2. 8월 21일: 스파로스 데브엑스 기반 프로젝트 구현 및 멘토링
3. 8월 28일: 8시간 템플릿 기반 발표 패키징, 팀별 발표, 회고, 현업 적용 인사이트 도출

## 운영 원칙

- 사전 설문과 참가자 명단은 공개 페이지에 원문으로 노출하지 않고 집계 인사이트만 반영합니다.
- 15명은 3인 1팀, 총 5개 프로젝트팀으로 구성하고 각 팀은 업무 오너, 구현 담당, 검증·발표 담당을 둡니다.
- 팀 주제는 서비스 운영, 업무 시스템, 클라우드/인프라, AX/R&D, 품질/준법/프로젝트 관리 축에서 실제 구현 가능한 현업 문제로 좁힙니다.
- 개별 외부 유료 AI 툴 결제를 전제로 하지 않고 사내 AI 플랫폼인 스파로스 데브엑스를 기본 실습 환경으로 둡니다.
- 결과물은 단순 학습 완료가 아니라 현업에서 어떤 업무를 어떻게 처리할지에 대한 실행 인사이트입니다.
