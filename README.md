# Shinsegae I&C AI Native Workshop

서울대학교 EPM 과정에서 2026년 6월 26일 오후 진행하는 신세계 I&C 대상 `AI Native 조직` 강의용 GitHub Pages입니다.

## 구성

- `index.html`: 강의 허브 단일 페이지
- `assets/styles.css`: 반응형 페이지 스타일
- `assets/shinsegae-seminar-cover.png`: 6/26 강의자료 표지 이미지
- `outputs/seminar-20260626.pdf`: 강의자료 PDF

1. AI 기술의 속도와 방향
2. AI Native & AI Harness 적용 사례
3. 우리 업무에 어떻게 도입할 수 있을까
4. 프롬프트/컨텍스트/하네스 엔지니어링
